ProxyPush app

ProxyPush is an application to help people behind a proxy which requires authentication. You need to sign in only once with this application, and all requests by other applications will be automatically authenticated to the remote proxy server. You need to set this as system proxy to work. It is portable so installation and administrative privileges are not required.
Expected release: 2020 February
